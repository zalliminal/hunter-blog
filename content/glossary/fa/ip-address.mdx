## آدرس <span dir="ltr">IP</span> چیه؟

خلاصه در یه جمله: <span dir="ltr">IP</span> <span dir="ltr">Address</span> مثل آدرس خونه‌ته تو دنیای اینترنت — هر دستگاهی یه آدرس منحصر به فرد داره که بقیه بتونن پیداش کنن.

بذار ساده بگم. وقتی می‌خوای برای کسی نامه بفرستی، باید آدرسش رو بدونی درسته؟ تو اینترنت هم همینطوره. هر دستگاهی (کامپیوتر، موبایل، سرور، حتی یخچال هوشمند!) یه آدرس عددی داره که بهش می‌گن <span dir="ltr">IP</span> <span dir="ltr">Address</span>. بدون این آدرس، هیچکس نمی‌تونه داده‌ها رو بفرسته برات.

### دو نسخه <span dir="ltr">IP</span> داریم:

**<span dir="ltr">IPv4</span>:** قدیمی‌تره، از چهار تا عدد تشکیل شده مثل `__PROTECTED_6__.__PROTECTED_7__.__PROTECTED_8__.__PROTECTED_9__`. هر عدد بین <span dir="ltr">۰</span> تا <span dir="ltr">۲۵۵</span> می‌تونه باشه. مشکلش اینه که کل <span dir="ltr">IPv4</span> فقط حدود <span dir="ltr">۴</span> میلیارد آدرس داره که دیگه کم اومده .

**<span dir="ltr">IPv6</span>:** جدیدتره، از اعداد و حروف تشکیل شده مثل `__PROTECTED_13__:0db8:85a3:__PROTECTED_14__:__PROTECTED_15__:8a2e:__PROTECTED_16__:__PROTECTED_17__`. این نسخه خیلی خیلی آدرس بیشتر داره و مشکل کمبود آدرس رو حل می‌کنه.

### <span dir="ltr">IP</span> عمومی و خصوصی:

**<span dir="ltr">IP</span> عمومی:** آدرسیه که اینترنت می‌بینه. مثل آدرس خونه‌ت که پستچی می‌بینه. این آدرس رو <span dir="ltr">ISP</span> (شرکت اینترنتت) بهت می‌ده.

**<span dir="ltr">IP</span> خصوصی:** آدرسیه که فقط تو شبکه داخلی خونه یا شرکتت استفاده می‌شه. مثل `__PROTECTED_22__.__PROTECTED_23__.__PROTECTED_24__.__PROTECTED_25__` که معمولاً روی مودم‌ها می‌بینی. این آدرس‌ها تو اینترنت قابل دسترسی نیستن.

### چرا برای امنیت مهمه؟

چون مهاجمان می‌تونن از <span dir="ltr">IP</span> برای شناسایی موقعیت تقریبی تو، اسکن کردن پورت‌هات، یا حتی حملات <span dir="ltr">DDoS</span> استفاده کنن. همچنین <span dir="ltr">IP</span> <span dir="ltr">Spoofing</span> وجود داره که مهاجم <span dir="ltr">IP</span> خودش رو جعل می‌کنه تا هویتش مخفی بمونه.
