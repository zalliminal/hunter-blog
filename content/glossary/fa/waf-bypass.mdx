## <span dir="ltr">WAF</span> <span dir="ltr">Bypass</span> چیه؟

خلاصه در یه جمله: <span dir="ltr">WAF</span> <span dir="ltr">Bypass</span> یعنی پیدا کردن راهی که نگهبان (<span dir="ltr">WAF</span>) رو گول بزنی و برسی به داخل مهمونی (سرور)!

یادته گفتم <span dir="ltr">WAF</span> مثل نگهبانه؟ خب مهاجمان زرنگ‌ترن! اونا یاد گرفتن چطور نگهبان رو دور بزنن. مثلاً نگهبان می‌گه "چاقو ممنوع!"، مهاجم چاقو رو لای روزنامه می‌پیچه و می‌گه "این فقط روزنامه‌ست!" و نگهبان هم باور می‌کنه .

### تکنیک‌های رایج <span dir="ltr">WAF</span> <span dir="ltr">Bypass</span>:

**<span dir="ltr">۱</span>. <span dir="ltr">Encoding</span> (کدگذاری):**
به جای `<__PROTECTED_10__>` می‌نویسی `%__PROTECTED_11__%__PROTECTED_12__`. بعضی WAFها این رو نمی‌شناسن!

**<span dir="ltr">۲</span>. <span dir="ltr">Case</span> <span dir="ltr">Toggling</span> (تغییر حروف):**
به جای `__PROTECTED_16__` می‌نویسی `__PROTECTED_17__`. بعضی WAFها فقط حروف بزرگ رو چک می‌کنن!

**<span dir="ltr">۳</span>. <span dir="ltr">Comment</span> <span dir="ltr">Injection</span> (تزریق کامنت):**
به جای `__PROTECTED_21__` می‌نویسی `__PROTECTED_22__/**/__PROTECTED_23__`. <span dir="ltr">WAF</span> فکر می‌کنه دو تا کلمه‌ست، ولی دیتابیس می‌فهمه یکیه!

**<span dir="ltr">۴</span>. <span dir="ltr">HTTP</span> <span dir="ltr">Parameter</span> <span dir="ltr">Pollution</span>:**
همون پارامتر رو چند بار می‌فرستی. <span dir="ltr">WAF</span> گیج می‌شه کدوم رو چک کنه!

**<span dir="ltr">۵</span>. <span dir="ltr">Path</span> <span dir="ltr">Traversal</span> <span dir="ltr">Tricks</span>:**
به جای `/__PROTECTED_34__/__PROTECTED_35__` می‌نویسی `/....//....//__PROTECTED_36__/__PROTECTED_37__`. بعضی WAFها این رو نمی‌شناسن!

**<span dir="ltr">۶</span>. <span dir="ltr">HTTP</span> <span dir="ltr">Verb</span> <span dir="ltr">Tampering</span>:**
به جای <span dir="ltr">GET</span> از <span dir="ltr">POST</span> استفاده می‌کنی یا برعکس. بعضی WAFها فقط یه روش رو چک می‌کنن .

### چرا برای امنیت مهمه؟

چون نشون می‌ده <span dir="ltr">WAF</span> به تنهایی کافی نیست! باید کدتم امن باشه. همچنین اگه بدونی چطور <span dir="ltr">WAF</span> دور زده می‌شه، می‌تونی <span dir="ltr">WAF</span> بهتری کانفیگ کنی که این ترفندها رو بشناسه.
